# demo-clips/

Drop short industrial fire/smoke videos here for the demo (.mp4 preferred).

Easiest source: edit the `URLS` list in `scripts/download_sample_clips.py` with 3–5
YouTube URLs of industrial fire footage and run it. Or curl/copy your own clips.

Filename convention: `clip_001.mp4`, `clip_002.mp4`, etc. — `service/config.py` defaults
to `clip_001.mp4` as the source.

This directory's contents are gitignored except for this README.
